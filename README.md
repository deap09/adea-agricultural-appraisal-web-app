# Adea's Agricultural Appraisal Web App

https://adea-agricultural-appraisal.web.app/

## Run locally

```js
npm install
```

```js
npm run dev
```

## Tech stack

### Frontend

- React
- TypeScript
- Redux
- Mapbox
- TailwindCSS

### Backend (Serverless)

- NoSQL Database: Firestore
- Hosting: Firebase Hosting
